# Day 4 – Alerts vs Incidents

## Learn

### Alert

Triggered by **detection rule**.

### Incident

One or more **alerts grouped together**.

### Why Grouping Exists

* alert fatigue reduction
* correlation across sources

### Understand Entity Mapping

Entities:

* User
