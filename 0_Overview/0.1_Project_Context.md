# Sample Detection Query

Example: **Brute Force Login Detection**

```
SigninLogs
| where ResultType != 0
| summarize FailedAttempts = count() by IPAddress, bin(TimeGenerated,5m)
| where FailedAttempts > 10
```

This query identifies **multiple failed authentication attempts from the same IP within a short time window**.
